Diet Unquiet Exercise API Documentation

Our restful API has been custom built to fit every individuals needs.
We have our exercise API custom built to look into any type of information
one might be looking for when it comes to exercises

Overview Commands

This initial section will have a list of all the different API endpoints the
user would be able to use in order to use the api. In the ongoing sections the
enpoints will be discussed in further detail.

The endpoint to get the exercises for one body part:

https://vast-teal-ostrich-ring.cyclic.app/Arms

The endpoint to get a specific exercise:

https://vast-teal-ostrich-ring.cyclic.app/Arms/0

The endpoint to get all exercises:

https://vast-teal-ostrich-ring.cyclic.app/

How to Use The Body Categorization Endpoint

The endpoint to get the exercises for one body part:

https://vast-teal-ostrich-ring.cyclic.app/Arms

The API is seperated by four body parts. These body parts are Legs, Arms, Chest,
and Back By adding the name of one of these body parts to the end of the URL, all
the exercises pertaining to said body part would display.

How to Call a Specific Exercise Using The Endpoint

The endpoint to get a specific exercise:

https://vast-teal-ostrich-ring.cyclic.app/Arms/0

The API is seperated by four body parts. These body parts are Legs, Arms, Chest,
and Back. Inside of each of these categories, there are also 25 exercises. Arms contains
exercises 0-24, Legs contains 25-49, Chest contains 50-74, and Back contains 75-99.
Putting a number after the body part will give the desired exercise.

How to List All The Exercises Using The Endpoint

The endpoint to get all exercises:

https://vast-teal-ostrich-ring.cyclic.app/

By typing the URL without any specification, we are able to get the entire api content.
By doing this, we get to see all the exercises included into the api, and in which category
each exercise belongs.
